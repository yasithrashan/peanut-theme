# Peanut Theme

A warm, comfortable VS Code theme with carefully selected peanut-inspired colors. Available in both light and dark variants for a smooth, distraction-free coding experience.

## Features

- **Two Beautiful Variants** - Choose between light and dark themes, both featuring the same warm peanut color palette
- **Easy on the eyes** - Warm tones and balanced contrast reduce fatigue during extended coding sessions
- **Thoughtful colors** - Each syntax element uses distinct, purposeful colors that make code easy to scan and understand at a glance
- **Clean interface** - Minimal UI styling keeps your focus on the code, not the chrome
- **Versatile** - Works beautifully across JavaScript, TypeScript, Python, HTML, CSS, Markdown, and more

## Installation

**From VS Code:**
1. Open Extensions (`Ctrl+Shift+X` or `Cmd+Shift+X`)
2. Search for "Peanut Theme"
3. Install and select your preferred variant from Color Theme preferences
   - `Peanut Light` for the light theme
   - `Peanut Dark` for the dark theme

**Manual:**
```bash
code --install-extension YasithRashan.peanut-theme
```

## Color Palette

Both themes share the same signature peanut color palette:
- **Orange Accent** (#FF8C42) - Used for highlights and active elements
- **Warm Browns** - Primary text and UI elements
- **Rich Syntax Colors** - Purple keywords, green strings, blue variables, red functions
